# Premier League Dashboard (Power BI)

## 📊 Project Overview
This project analyzes Premier League data using Power BI dashboards to derive insights on player performance, goals, assists, and team statistics.

## 📌 Features
- Player performance analysis
- Goals & assists visualization
- Team comparison dashboard
- Match trends analysis

## 🛠 Tools Used
- Power BI
- Data Visualization
- Data Cleaning
